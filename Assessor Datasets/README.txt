- `logs` contains the conversations we used to test the assessor
- `refs` contains the manually annotated conversation examples we used as ground truth.


The results for different versions are in the following three folders:

- `FULL` contains the assessments produced by the "Full Assessor".
- `NOJUST` contains the assessments produced by the "NO Justification" version of the Assessor.
- `SHORT` contains the assessments produced by the "Short Memory" version of the Assessor.
